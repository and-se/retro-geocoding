Зилов А.А. около 2014 года проработал тексты книг для букв А-К и расставил GPS координаты для небольших населённых пунктов (сёла, деревни), во многих случаях добавив полезные комментарии и исправления.
Для городов (тем паче крупных, в частности храмов и монастырей внутри них) данных нет.

В папке input лежат результаты преобразования предоставленных А.А. Зиловым .doc файлов в html.

Для начала работы html надо прогнать через preprocess_data.py





== Прочие ресурсы, откуда можно будет взять дополнительные сведения ==

=== martyrmap.ru ===

martyrmap.ru - интерактивная карта мест служения Новомучеников, разрабатываемая приходом храма Новомучеников и Исповедников Российсков в Бутово.

Содержит сведения в основном по Московской области.

При заходе на сайт открывается карта со всеми метками. Если поанализировать, какие запросы отправляются, то можно найти один любопытный запрос:

https://api-maps.yandex.ru/services/geoxml/1.2/geoxml.xml?callback=id_1586291884312526818&url=http%3A%2F%2Fmartyrmap.ru%2Fget_yml.php%3F1586291885278

- т.е. API yandex-карт даётся ссылка http://martyrmap.ru/get_yml.php?1586291885278

При обращении по этой ссылке получаем все метки, отображаемые на карте, в виде вполне читаемого XML, что пригодно для дальнейшей работы с этими данными.

Надо только конечно не забыть указать, откуда эти данные взяты!

Это не вся информация - есть дополнительная возможность получить список людей, связанных с меткой на карте. А также список меток на карте, связанных с человеком.
Если поисследовать, то последнее также делается тем же запросом с указанием параметра ID записи о человеке.


=== temples.ru ===

temples.ru

Проект «Храмы России» создан для сбора и систематизации документальных сведений о храмах, существовавших на территории нынешней Российской Федерации к 1917 году и построенных в постсоветское время. Проект посвящен православным и старообрядческим храмам, уничтоженным и сохранившимся до наших дней, забытым и действующим.

Есть возможность скачать GPS координаты храмов по каждому региону РФ:
1) http://temples.ru/tree.php
2) Выбираем регион
3) Внизу Файл для Google Earth, например http://temples.ru/getkml.php?TreeID=717
В этих данных GPS координаты, названия храмов, современные адреса, а также ссылки на страницу храма на сайте - на этой странице есть более подробная информация, в частности адрес на 1917 год.

Возможно, также применима техника как для martyrmap.ru, но вряд ли она даст больше сведений, чем описанный выше способ.
